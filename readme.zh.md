# Datapack Updater
[English](./readme.md) | **简体中文**

该项目旨在将您的命令从 1.20.4 升级到较新版本的 Minecraft (1.20.5)。

## 特点

**支持的命令**
 - [x] `/give`
 - [x] `/item`
 - [x] `/clear`
 - [x] `/summon` for 'Firework' and 'Item'
 - [ ] `/setblock` (Loot Table)
 - [ ] `/fill` (Loot Table)
 - [x] 选择器 (以 `@` 开头，比如 `@a[nbt={...}]`)
 - [ ] Loot Table
 - [ ] Item modifier
